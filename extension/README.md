# Boss直聘AI投递助手

🤖 一个智能浏览器插件，自动分析职位匹配度并一键投递，让您的求职之旅更加高效！

## ✨ 功能特性

- **智能匹配度分析**：根据您的技能和求职意向自动计算职位匹配度
- **视觉匹配指示器**：在职位卡片上直观显示匹配度百分比
- **一键自动投递**：自动完成沟通、打招呼全流程
- **自定义配置**：支持设置技能、职位意向、自我介绍模板等
- **投递统计**：记录匹配岗位数、已投递数、已回复数

## 📦 安装方法

### 方式一：从Chrome网上应用店安装（即将上线）

### 方式二：手动安装（开发者模式）

1. 下载或克隆本仓库
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本项目的 `extension` 文件夹

## 🚀 使用方法

1. 打开 [Boss直聘](https://www.zhipin.com) 网站
2. 搜索您感兴趣的职位
3. 点击浏览器右上角的插件图标（紫色"B"图标）
4. 在弹出窗口中配置：
   - **我的技能**：选择或输入您的技能标签
   - **求职意向**：设置目标职位、工作经验、期望薪资
   - **自我介绍模板**：编辑您的自我介绍内容
   - **投递设置**：设置匹配度阈值
5. 点击 **🚀 开始投递**，插件将自动完成投递流程

## 🔧 配置说明

### 技能选择
- 支持预设技能标签点击选择
- 支持自定义输入其他技能（用逗号分隔）

### 自我介绍模板
- 使用 `{skills}` 占位符，将自动替换为您选择的技能列表
- 支持换行和个性化内容

### 匹配度阈值
- 默认60%，低于此阈值的职位将被跳过
- 绿色（≥80%）、黄色（60-79%）、红色（<60%）
